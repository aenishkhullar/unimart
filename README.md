# UniMart 🛒

A full-stack MERN marketplace platform designed for university students to buy, sell, and rent products within their campus ecosystem.

## 🚀 Features

- User Authentication (Login/Register)
- Create, Edit, Delete Products
- Buy & Rent functionality
- Protected Routes
- Product Details Page
- Ownership-based access control

## 🛠 Tech Stack

- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT

## 📁 Project Structure

client/ → React frontend  
server/ → Node.js backend  
models/ → Database schemas  
routes/ → API routes  
controllers/ → Business logic 

## ⚙️ Installation

### Clone the repository
git clone https://github.com/your-username/unimart.git

### Backend setup
cd server
npm install
npm run dev

### Frontend setup
cd client
npm install
npm run dev
